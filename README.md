
# Systemaide-Accounting
A web-based accounting system.

## Install Dependencies

For Backend

```bash
  cd backend
  npm install
```

For Frontend

```bash
  cd frontend
  npm install
```
## Run Locally

Clone the project

```bash
  git clone https://github.com/RusselDelaCruz27/Systemaide-Accounting.git
```

Go to the project directory

```bash
  cd Systemaide-Accounting
```

Start the Frontend

```bash
  cd frontend
  npm run dev
```
